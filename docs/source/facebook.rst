Facebook - Messenger
####################

This Facebook API when given a name and message, this bot will message automatically in Facebook. 

.. tabs::

   .. code-tab:: py

        #pip install bot_studio
        from bot_studio import *
        dk=bot_studio.new()
        dk.facebook__messenger(Send_message_to='mutyalu ',Message='hi')
